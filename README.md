#  JWT Authentication Mini Project

A full-stack authentication project using Node.js, Express.js, MongoDB, JWT and React (Vite).  
This project demonstrates secure user authentication with signup, login, and protected routes.


##  Features

- User Signup & Login
- Password hashing using bcrypt
- JWT-based authentication
- Protected API routes
- RESTful API design
- Frontend built with React + Vite
- Backend built with Node.js & Express
- MongoDB Atlas integration



##  Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB (Atlas)
- JWT (JSON Web Token)
- bcryptjs
- dotenv

### Frontend
- React.js
- Vite
- Axios
- CSS


##  Project run

### Frontend

npm i
cd features
npm run dev


### Backend

cd back
npm i
node server.js
