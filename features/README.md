# JWT Authentication Mini Project (MERN)

This is a beginner-friendly JWT authentication project built using
React (Vite) for frontend and Node.js + Express + MongoDB for backend.



##  Features

- User Signup
- User Login
- Password Hashing using bcrypt
- JWT Token generation
- Protected Profile API
- Token stored in browser LocalStorage
- Logout functionality



##  Tech Stack

### Frontend
- React (Vite)
- JavaScript
- Fetch API

### Backend
- Node.js
- Express.js
- MongoDB (Atlas)
- JSON Web Token (JWT)
- bcryptjs
- dotenv
- cors




##  How to Run the Project ??

### Frontend

1. npm i
2. cd features
3. npm run dev

### Backend

1. cd back
2. npm i
3. node server.js
